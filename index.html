Fórmulas y Propiedades

Operaciones:

1. B + C

2. A(B + C)D

3. (c₁A)((c₂B + c₃C))(c₄D)

Propiedades:

  • Distributividad: A(B + C) = AB + AC
  • Asociatividad con escalares: c(AB) = (cA)B = A(cB)
  • Las matrices deben tener dimensiones compatibles

Tipos de números:

  • Enteros (ℤ): {..., -2, -1, 0, 1, 2, ...}
  • Racionales (ℚ): números expresados como p/q
  • Reales (ℝ): incluye todos los números decimales
